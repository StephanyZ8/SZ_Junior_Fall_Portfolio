Holiday Drawing

For this project I was able to create a festive christmas scene by using Python and its SimpleGUI canvas to mimic a decorated tree, Santa Claus, colorful (animated) lights, and a holiday banner. I manipulated loops and lists to efficiently animate repeating elements like the colorful lights to make the scene more dynamic and visually interesting each time it runs. This project helped me better understand how to work with coordinates, shapes, and drawings in order to build detailed drawings with python. Although positioning everything correctly was challenging at first, adjusting my values through experimentation improved my problem solving skills and ultimately boosted confidence in coding.
